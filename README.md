# Smart VPN Panel

Умная VPN-система на базе AmneziaWG и sing-box-extended с автоматической балансировкой и встроенной панелью управления.

## Установка

Для установки всей системы (включая веб-панель и оркестратор) на свежий сервер **Ubuntu 22.04**, выполните всего одну команду от имени `root`:

```bash
wget -qO- https://raw.githubusercontent.com/blablajka/serv/main/deploy.sh | bash
```

Эта команда:
1. Автоматически скачает все нужные скрипты из этого репозитория.
2. Установит необходимые зависимости и создаст Swap-файл (чтобы сервер не завис).
3. Скомпилирует ядра `awg-server` и `sing-box`.
4. Настроит маршрутизацию (tun).
5. Запустит красивую веб-панель.

В случае возникновения ошибок, скрипт выделит их красным цветом и покажет строку, на которой произошел сбой.

## Доступ к панели

После успешной установки веб-интерфейс будет доступен по адресу:
`http://ВАШ_IP_АДРЕС:5000`

- **Логин:** admin
- **Пароль:** admin
